Heutzutage gibt es viele Arten von Geräten, die auf das Web zugreifen. 
Sie reichen von Desktop-Computern mit großen Monitoren bis zu Smartphones 
mit kleinen Displays. 
Diese Geräte haben unterschiedliche Bildschirmgrößen, Auflösungen und 
Verarbeitungsleistungen. 
Responsive Web Design ist ein Ansatz zum Entwerfen von Webinhalten, 
der auf die Einschränkungen verschiedener Geräte reagiert. 
Die Seitenstruktur und die CSS-Regeln sind flexibel, 
um diesen Unterschieden Rechnung zu tragen. CSS bietet die Werkzeuge, 
um verschiedene Stilregeln zu schreiben und die dann abhängig vom Gerät anzuwenden, 
auf dem die Seite angezeigt wird.

## Responsive Webdesign

### Media Querys

Media Querys sind eine Technik, die in CSS3 eingeführt wurde und die Darstellung von Inhalten basierend auf unterschiedlichen Ansichtsfenstergrößen ändert. Das Ansichtsfenster ist der sichtbare Bereich eines Benutzers auf einer Webseite und hängt vom Gerät ab, mit dem auf die Site zugegriffen wird.

Medienabfragen bestehen aus einem Medientyp. Wenn dieser mit dem Gerätetyp übereinstimmt, auf dem die Website angezeigt wird, werden die Stile angewendet. 

Hier ist ein Beispiel für eine Media Query, die den Inhalt zurückgibt, 
wenn die Breite des Geräts kleiner oder gleich 350 Pixel ist:

```
@media (max-width: 350px) { /* CSS Regeln */ }

```

Die folgende Media Query gibt den Inhalt zurück, wenn die Höhe des Geräts mindestens 350 Pixel beträgt:

```
@media (min-width: 350px) { /* CSS Regeln */ }

```

> Denke daran, dass das CSS in der Medienabfrage nur angewendet wird, wenn der Medientyp mit dem des verwendeten Geräts übereinstimmt.






### Bilder

Es ist eigentlich sehr einfach, Bilder mit CSS ansprechbar zu machen. 
Sie müssen einem Bild nur folgende Eigenschaften hinzufügen:


```
img {
  max-width: 100%;
  height: auto;
}

```

Die maximale Breite von 100% stellt sicher, dass das Bild niemals breiter als 
der Container ist, in dem es sich befindet, und die Höhe von `auto` bewirkt, dass das Bild sein ursprüngliches Seitenverhältnis beibehält.

### Retina-Display

Mit der Zunahme von Geräten mit Internetanschluss variieren ihre Größen 
und Spezifikationen, und die von ihnen verwendeten Displays können extern 
und intern unterschiedlich sein. 
Die Pixeldichte ist ein Aspekt, der sich auf einem Gerät von anderen unterscheiden 
kann. Diese Dichte wird als Pixel pro Zoll (PPI) oder Punkte pro Zoll (DPI) 
bezeichnet. 
Das bekannteste Display dieser Art ist das sogenannte "Retina Display" auf 
den neuesten Apple MacBook Pro-Notebooks und kürzlich auf iMac-Computern. 
Aufgrund des Unterschieds in der Pixeldichte zwischen einer "Retina" - und 
einer "Nicht-Retina" -Anzeige können einige Bilder, die nicht mit einer 
hochauflösenden Anzeige erstellt wurden, beim Rendern auf einer hochauflösenden 
Anzeige "pixelig" aussehen.

Die einfachste Möglichkeit, Ihre Bilder auf hochauflösenden Displays wie 
dem "Retina-Display" des MacBook Pros korrekt anzuzeigen, besteht darin, 
ihre Werte für Breite und Höhe als nur die Hälfte der ursprünglichen Datei 
zu definieren. 
Hier ist ein Beispiel für ein Bild, das nur die Hälfte der ursprünglichen 
Höhe und Breite verwendet:

```
<style>
  img { height: 250px; width: 250px; }
</style>
<img src="coolPic500x500" alt="A most excellent picture">


```
### Typografie

Anstatt `em` oder `px` zum Formatieren von Text zu verwenden, 
können wir Ansichtsfenstereinheiten für reaktionsschnelle Typografie verwenden. 
Ansichtsfenstereinheiten sind wie Prozentsätze relative Einheiten, 
basieren jedoch auf unterschiedlichen Elementen. 
Ansichtsfenstereinheiten beziehen sich auf die Ansichtsfensterabmessungen 
(Breite oder Höhe) eines Geräts, und Prozentsätze beziehen sich auf die Größe 
des übergeordneten Containerelements.

Die vier verschiedenen Einheiten sind:

- vw (Breite des Ansichtsfensters): 10vw entspricht 10% der Breite des Ansichtsfensters.
- vh (Ansichtsfensterhöhe): 3vh entspricht 3% der Höhe des Ansichtsfensters.
- vmin (Minimum des Ansichtsfensters): 70vmin entspricht 70% der kleineren Abmessung des Ansichtsfensters (Höhe oder Breite).
- vmax (maximales Ansichtsfenster): 100 vmax entspricht 100% der größeren Abmessung des Ansichtsfensters (Höhe oder Breite).

In diesem Beispiel wird ein Body-Tag auf 30% der Breite des Ansichtsfensters festgelegt.


```
body { width: 30vw; }
```
